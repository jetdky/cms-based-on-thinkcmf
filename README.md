### 注意
英文分类不要和中文分类混合添加    
同样中文内容不要与英文内容混合添加    
混合添加分类/内容不会显示
* 规范一 英文分类用英文名，中文分类用中文名
* 规范二 新增加内容/分类前，先选择语言（分类根据语言变化） 
* 规范三 单页分类中不添加内容，内容都添加到内容管理中
* 规范四 所有单页一级分类下至少有一个二级分类（banner），且内容**不能直接**添加在单页 的一级分类下
